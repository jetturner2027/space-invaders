# Space Invaders

A Space Invaders clone built in Java. Based on the original 1978 arcade edition of the classic game. Includes leaderboard that saves between launches, full gameplay in a Draw window.

## How to Run

Download the latest release and run:
'''java -jar SpaceInvaders.jar'''

Make sure `leaderboard.txt` is in the same folder as the JAR file.

Full setup with .jar file and .txt file is included in the 'Space Invaders.zip' file.

## Features

- Classic Space Invaders gameplay
- Persistent leaderboard saved in a text file
- Arcade Graphics in a Draw window

## Built With

- Java
- IntelliJ IDEA

## How to Build from Source

Open the project in IntelliJ IDEA, then go to Build → Build Artifacts to generate the JAR. Otherwise, a built version is included in the .zip
